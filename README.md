# 🔄 Conversor de Unidades

Um projeto simples em **Python** para converter diferentes unidades de medida diretamente no terminal. 

Funcionalidades
- 🌡️ **Temperatura**: Celsius ↔ Fahrenheit  
- 🚗 **Distância**: Km ↔ Milhas  
- ⚖️ **Peso**: Kg ↔ Libras
